发表于 2017-08-08 | 分类于 Demo |

前几天学姐跟我讨论了一道题，觉得还挺有意思的。

题目描述：
请使用原生 JavaScript 实现一个方法，判断 html 中出现次数最多的标签，并统计这个次数。

解题思路：

  1. 取到页面中所有标签，这里是会有重复的；
  2. 遍历所有标签，统计每个标签出现的次数。
